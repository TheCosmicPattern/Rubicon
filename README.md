Note: run_isw_test_v10_signed.py requires the Planck SMICA CMB maps and DESI DR1 galaxy catalogs (approx. 50GB of FITS files). 
You can download the CMB maps from the ESA Planck Legacy Archive (https://pla.esac.esa.int) and the galaxy catalogs from the DESI Data Release 1 (DR1) portal 
(https://data.desi.lbl.gov/doc/releases/dr1/).
